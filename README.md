# Campaign Performance Analytics

## Project Overview

This project analyzes marketing campaign performance using SQL Server and Excel. The goal is to evaluate campaign effectiveness and identify opportunities to improve customer engagement, conversions, and revenue.

## Dataset

The project uses four tables:

* Customers
* Campaigns
* Campaign Interactions
* Conversions

## Key Metrics

* Click-Through Rate (CTR)
* Conversion Rate
* Revenue by Campaign
* ROI
* Customer Segment Performance
* Acquisition Channel Analysis
* Funnel Analysis

## Tools Used

* SQL Server
* Microsoft Excel
* Pivot Tables
* Charts and Dashboards

## Key Findings

* Identified campaigns with the highest engagement and revenue.
* Analyzed customer segments and acquisition channels.
* Evaluated campaign ROI and conversion performance.
* Performed funnel analysis to identify drop-off points.

## Business Recommendations

* Increase investment in high-performing campaigns.
* Optimize campaigns with high clicks but low conversions.
* Focus on customer segments with the highest revenue contribution.
